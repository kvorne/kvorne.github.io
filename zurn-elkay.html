    How might we unify the experience for sales reps within two merging companies?

    Overview

    Role: UX / UI Designer
    Team: Katarina Vorne, Sharon Thomas
    Timeline: 10 Weeks (40 hr/week)
    Tools: Figma, Microsoft Clarity, Maze User Testing
    View Prototype

    Context

    Zurn Industries and Elkay Manufacturing recently merged into Zurn-Elkay Water Solutions. Zurn used a Sales Enablement Platform called Highspot, while Elkay developed “RepMyElkay.” Each of these tools had the same purpose: to provide sales representatives with tools and information.

    Assignment

    Consolidate Zurn and Elkay’s individual solutions for providing order search and informational materials into a unified and intuitive experience.

    My Role

    I worked on this project as a User Experience and Interface Designer under the guidance of Sharon Thomas, Principal UX/UI Designer at Zurn-Elkay Water Solutions.

    My individual contributions included the following:
    • Style Guide
    • WCAG 2.0 AAA Accessibility Standards
    • User Experience of the Library and Main Menu
    • Usability Testing
    • Qualitative and Quantitative Data Analysis and Reporting
    I collaborated with Sharon Thomas on the following:
    • Design System
    • User Experience of the Order Search
    • Footer Design
